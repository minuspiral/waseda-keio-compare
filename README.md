# 早慶理工 入学者偏差値推定2025

慶應義塾大学理工学部・早稲田大学理工3学部の**入学者偏差値**を、公式入試統計と河合塾全統模試データ、駿台ベネッセの併願成功率から推定します。あくまで推定値なので留意ください。なお、早稲田については合格者平均分布が公表されている8学科のみで推定を行っています。
URLは**[https://minuspiral.github.io/waseda-keio-compare/]**です

## 概要

早慶理工の合格者平均偏差値は河合塾で集計されていますが、**実際に入学する学生の平均偏差値**は辞退者の影響で大きく異なります。本ツールは、併願行動・国立大学合否・浪人選択をモデル化し、入学者の偏差値分布を推定します。

## 主な分析内容

- **入学者構成分析**: 入学者を東大落ち・京大落ち・東工大落ち・第一志望等に分解し、各グループの推定人数と平均偏差値を表示
- **学科別比較**: 慶應5学門＋早稲田8学科の入学者平均偏差値を統一テーブルで比較（ソート対応）
- **学門/学科カード**: 各学門・学科ごとの合格者・入学者の偏差値分布をヒストグラムで表示
- **モデル検証**: 駿台ベネッセ併願成功率データ(2023)との整合性を検証

## データソース

| データ | 出典 |
|---|---|
| 合格者偏差値分布 | 河合塾 全統模試 偏差値帯別合格者分布 |
| ボーダー偏差値 | 河合塾 2026年度入試難易予想 |
| 合格者平均偏差値 | 慶應義塾大学・早稲田大学 2025年度公式入試統計 |
| 募集人数・合格者数 | 慶應義塾大学・早稲田大学 2025年度公式入試統計 |
| 併願成功率 | 駿台ベネッセ 2023年度データ |

## 推定モデル

辞退率を以下の3要素でモデル化しています:

1. **国立大学合格辞退**: 東大・京大・東工大との併願率と合格確率から推定
2. **早慶W合格辞退** (早稲田のみ): 慶應との同時合格時の選択率を反映
3. **浪人選択**: 偏差値帯別の浪人率を適用

各パラメータは公式統計の入学者数・合格者平均と整合するよう校正しています。

## 注意事項

- 本ツールの数値は統計モデルによる**推定値**であり、実際の値とは異なる可能性があります
- 個人の合否を予測するものではありません
- データソースの更新に伴い、推定値は変動する可能性があります
